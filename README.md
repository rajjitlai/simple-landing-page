# Simple Landing Page

A clean and modern landing page built with HTML and CSS featuring a glassmorphism design aesthetic.

![Landing Page Preview](design/design.png)

## Features

- 🎨 Modern glassmorphism UI design
- 📱 Responsive layout
- 🖼️ SVG assets for crisp visuals
- 🔗 Social media integration
- ⚡ Lightweight - No JavaScript frameworks required
- 🎯 Clean and semantic HTML structure

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Poppins)
- Unicons (Icon library)

## Getting Started

### Prerequisites

- A modern web browser
- A local server (optional, but recommended)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rajjitlai/simple-landing-page.git
```

2. Navigate to the project directory:
```bash
cd simple-landing-page
```

3. Open `index.html` in your browser or serve it using a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

4. Visit `http://localhost:8000` in your browser

## Customization

### Colors
The primary colors used in the design:
- Primary: `#362567` (Dark purple)
- Secondary: `#646afb` (Blue-purple)
- Accent: `#97b6f4` (Light blue)
- Text: `#2b2d69` (Dark blue)

### Fonts
- Family: Poppins (loaded from Google Fonts)
- Weights: 100-900 (all weights available)

### Social Links
Update the social media links in `index.html` (lines 22-25) to point to your profiles.
